Projeto Tela de Log in Minimalista

Esta tela foi desnevolvida com o intuito de trazer uma experiência mais minimalista, utlizando uma paleta de cores monocromáticas, o mesmo permite a nós vizualizar a tela sem muitas informações, o que muitas das vezes acaba sendo mais proveitosa a "User Experience" (UX).
